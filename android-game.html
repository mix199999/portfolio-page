Application Description

The "30 Seconds!" application was created for the Android system and is used for language learning. Users have the option to choose one of the five currently available languages, such as Polish, English, German, Italian, or Spanish. Within the game, four different types of tasks have been created, which the user must face. The application has a visually pleasing and user-friendly interface.

Technology Description

The following technologies were used to create the application:

  • Java
  • PHP
  • MySQL
  • SQLite

The application uses an API to retrieve and modify data from the database. However, a SQLite database is also locally created, which mirrors the database from the API. This decision was made to minimize server load and optimize application performance.

Application Architecture and Functionality Description

API

The application sends a request to the server, which then processes the request and sends feedback to the client in JSON format depending on the result. Then, the received data is processed in the client application.

Activities

The application architecture is based on three main activities, each of which has separate fragments.
